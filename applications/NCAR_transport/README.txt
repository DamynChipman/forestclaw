This applications directory shows the results of examples from the
NCAR Transport Workshop, held March 2011.

Details of the benchmarks can be found in

         P.H. Lauritzen, P.A. Ullrich, C. Jablonowski et al.,
              "A standard test case suite for two-dimensional
              linear transport on the sphere: Results from a collection
	      of state-of-the-art schemes", (submitted Aug. 2013).

         http://www.geosci-model-dev.net/recent_papers.html


Slotted disk
--------------
Scalar advection of a slotted-disk under a deforming wind field.
